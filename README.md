# 🌟 하루폴리오 (Harufolio) 서비스 소개

## 1. 하루폴리오란?

**하루폴리오(Harufolio)**는 **AI 상담원과 함께 만드는 초고속 웹사이트 제작 서비스**입니다.
복잡한 웹사이트 제작 과정을 AI와의 대화를 통해 간소화하여, 누구나 쉽고 빠르게 자신만의 웹사이트를 가질 수 있도록 돕습니다.

> **"하루 만에 완성하는 나만의 포트폴리오"**라는 의미를 담고 있으며, 합리적인 가격과 빠른 제작 속도를 핵심 가치로 삼고 있습니다.

---

## 2. 핵심 가치 (Core Values)

### 🚀 Speed (속도)

- AI가 24시간 대기하며 즉시 상담을 진행합니다.
- 불필요한 미팅 시간을 줄이고, 체계적인 프로세스로 제작 기간을 단축합니다.

### 💰 Affordability (합리성)

- **기본가 200,000원**이라는 파격적인 가격으로 시작할 수 있습니다.
- 꼭 필요한 기능만 담아 거품을 뺐으며, 추가 옵션은 투명하게 가격을 공개합니다.

### 🧩 Simplicity (간편함)

- 개발 지식이 없어도 AI에게 말하듯이 요구사항을 전달하면 됩니다.
- 디자인과 콘텐츠 수정 단계가 명확히 분리되어 있어 혼란 없이 진행됩니다.

---

## 3. 서비스 프로세스 (5단계)

하루폴리오는 체계적인 5단계 프로세스를 통해 완성도 높은 결과물을 만들어냅니다.

### 1️⃣ DEMO (초기 상담)

- **목적**: 웹사이트 제작 목적과 방향성 설정
- **내용**: AI가 방문자 타겟, 참고하고 싶은 사이트, 주요 기능 등을 질문하여 기획안을 완성합니다.

### 2️⃣ FIRST_REVIEW (디자인 확정)

- **목적**: 웹사이트의 뼈대와 스타일 결정
- **내용**: 레이아웃, 색상, 폰트 등 **디자인 요소**를 집중적으로 검토하고 수정합니다.
- _특징: 이 단계에서 디자인을 확정하면 이후에는 디자인 변경이 어렵습니다._

### 3️⃣ SECOND_REVIEW (콘텐츠 채우기)

- **목적**: 실제 들어갈 내용 완성
- **내용**: 텍스트(문구), 이미지, 프로필 정보 등 **콘텐츠**를 수정하고 교체합니다.
- _특징: 디자인은 유지한 채 내용물만 알차게 채워넣는 단계입니다._

### 4️⃣ EXTRA (추가 옵션 상담)

- **목적**: 더 풍성한 기능 추가 (선택 사항)
- **내용**: 섹션 추가, 애니메이션 효과, 디자인 전면 재작업 등 기본 제공 범위 밖의 기능을 원할 경우 상담합니다.
- _특징: 추가 비용이 발생하며, AI가 즉시 견적을 안내해 드립니다._

### 5️⃣ FINAL (최종 완료)

- **목적**: 최종 확인 및 전달
- **내용**: 모든 수정사항이 반영된 최종 결과물을 확인하고 제작을 완료합니다.

---

## 4. 가격 정책 (Pricing)

하루폴리오는 **정찰제**를 원칙으로 하여 예상치 못한 추가 비용 걱정을 덜어드립니다.

- **기본 제작비**: **200,000원** (KRW)
- **무료 수정 범위**:
  - 1차(디자인), 2차(콘텐츠) 각 단계별 수정 제공
  - 이미지 교체 10장까지 무료
  - 텍스트 전체 수정 1회 무료
- **유료 옵션 예시**:
  - 섹션 추가: 50,000원/개
  - 고급 디자인/애니메이션: 별도 견적

---

## 5. AI 상담원의 역할

하루폴리오의 AI는 단순한 챗봇이 아닌 **전문 웹 기획자**의 역할을 수행합니다.

- **전문적인 가이드**: 고객이 놓칠 수 있는 기획 요소를 먼저 질문하고 제안합니다.
- **명확한 범위 설정**: 수정 가능한 범위와 불가능한 범위를 명확히 안내하여 프로젝트 지연을 막습니다.
- **실시간 견적**: 추가 요청 사항에 대해 즉각적으로 비용을 산출하여 의사결정을 돕습니다.
- **기억과 관리**: 모든 대화 내용을 기억하여 중복 설명을 요구하지 않으며, 상담 내용을 체계적으로 정리하여 제작팀에 전달합니다.
